Probando el widget desde el deployment de Vercel

📍 Tu URL del Widget

1. Ve a tu Dashboard de Vercel y copia tu URL:

https://TU-PROYECTO-VERCEL.vercel.app

2. Tu widget estará disponible en:

https://TU-PROYECTO-VERCEL.vercel.app/segurbot-widget.umd.js

3. Prueba que funciona visitando esa URL directamente

Debería mostrarte el código JavaScript del widget, no un error 404.

⚠️ Configuración Requerida

  1. Actualiza las URLs en el código de abajo con tu URL real de Vercel
  2. Asegúrate de que el backend esté funcionando
  3. Verifica CORS en tu API para permitir tu dominio de Vercel
🚀 Ver Vercel Dashboard

✅ ¡El widget debería aparecer abajo a la derecha!

Si ves el botón flotante con el icono de chat, ¡el widget está funcionando correctamente online!

🔧 Código de Integración

Una vez que confirmes que funciona, usa este código en cualquier sitio web:

<!-- Reemplaza con tu URL real de Vercel --> <script src="https://TU-PROYECTO-VERCEL.vercel.app/segurbot-widget.umd.js" data-api-url="https://tu-backend.com/api/chat" data-title="Mi SegurBot" data-message="¡Hola! ¿En qué puedo ayudarte?" ></script>

📋 Checklist de Deploy

  • ☐ El proyecto está desplegado en Vercel
  • ☐ El archivo segurbot-widget.umd.js está en el build
  • ☐ El backend está funcionando y accesible
  • ☐ CORS está configurado para tu dominio de Vercel
  • ☐ La URL del widget responde correctamente
