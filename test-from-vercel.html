🚀 ¡Probando SegurBot desde Vercel!

✅ Tu configuración:

src="https://chatbotfrontend-tau.vercel.app/segurbot-widget.umd.js" data-api-url="http://localhost:8080/api/chat" data-title="Mi SegurBot" data-message="¡Hola! ¿En qué puedo ayudarte?"

🎯 ¿Qué debería pasar?

  1. Botón flotante: Debería aparecer un botón con icono en la esquina inferior derecha
  2. Sin texto: Solo el icono, manteniendo el diseño original
  3. Al hacer clic: Se abre la ventana de chat
  4. Mensaje personalizado: "¡Hola! ¿En qué puedo ayudarte?"

⚠️ Si no aparece el botón:

  1. Abre DevTools (F12) → pestaña Console
  2. Busca errores en rojo
  3. Verifica que tu backend esté corriendo en localhost:8080
  4. Prueba abrir esta URL directamente

📋 Para usar en otros sitios:

Solo copia y pega este código antes del </body>:

<script src="https://chatbotfrontend-tau.vercel.app/segurbot-widget.umd.js" data-api-url="https://tu-backend-real.com/api/chat" data-title="Mi SegurBot" data-message="¡Hola! ¿En qué puedo ayudarte?" ></script>
